Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are several methods to select elements within the Document Object Model (DOM):
1. document.getElementById(): Selects a single element by its id.
2. document.getElementsByClassName(): Selects all elements with a class name.
3. document.getElementsByTagName(): Selects all elements with a specified tag name.
4. document.querySelector(): Selects the first element that matches a specified CSS selector.
5. document.querySelectorAll(): Selects all elements that match a specified CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets the HTML markup contained within an element.
innerText: Retrieves or sets the visible text content of an element.
textContent: Accesses all text within an element.

Question-3: What is event delegation in the DOM?

In the DOM, event delegation is a technique where you attach a single event listener to a parent element instead of adding listeners to multiple child elements, allowing the parent to handle events that occur on its children.

Question-4: What is event bubbling in the DOM?

Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

1.To create a new HTML element:
const newElement = document.createElement('div');
2.To add the newly created element to the DOM:
const parentElement = document.getElementById('parent');
parentElement.appendChild(newElement);
3.To remove an element:
const elementToRemove = document.getElementById('elementId');
elementToRemove.remove();
