Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are several methods to select elements within the Document Object Model (DOM):
1. document.getElementById(): - Selects a single element by its unique `id` attribute.
2. document.getElementsByClassName(): - Selects all elements with a specified `class` name.
3. document.getElementsByTagName(): - Selects all elements with a specified tag name.
4. document.querySelector(): - Selects the first element that matches a specified CSS selector.
5. document.querySelectorAll(): - Selects all elements that match a specified CSS selector.
